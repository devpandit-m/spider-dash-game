# 🕷️ Spider Dash | Neon Portal Game

A stylish and interactive 2D web-based game where players must navigate a spider through obstacles. The project features a creative "Lamp-Pull" shutter mechanism for login and registration.


## ✨ Key Features
- **Interactive Login Portal:** Pull the lamp cord to reveal the login/signup shutter.
- **Secure Authentication:** Built-in login logic using `LocalStorage` to prevent direct access to the game.
- **Dynamic Gameplay:**
  - Speed increases as the score goes up.
  - **Day & Night Cycle:** Environment shifts every 10 points.
  - **Collision Detection:** Precise hitbox logic for birds (obstacles).
- **Responsive UI:** Smooth CSS3 animations and neon-themed design.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Keyframes & Flexbox)
- **Logic:** Vanilla JavaScript (RequestAnimationFrame & LocalStorage)

## 📁 Folder Structure
- `login.html` - The entry portal.
- `game.htm` - Main game screen.
- `sty.css` - Styles for the login portal.
- `style.css` - Styles for the game mechanics.
- `dev1.js` - Authentication & UI logic.
- `script.js` - Game engine & physics.

## 🔧 How to Run
1. Clone the repository: `git clone https://github.com`
2. Open `login.html` in any modern web browser.
3. Pull the lamp cord to start!
